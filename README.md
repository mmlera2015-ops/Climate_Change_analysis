# Climate_Change_analysis
# Climate Change Scenario Analysis

Climate change analysis project: Mann-Kendall & Sen trend analysis, temperature scenario plotting with Python & GIS

## Overview
- Climate change scenario analysis using CMIP6 data
- Trend detection and visualization of temperature & precipitation
- Analysis performed in Python
- Provides clear results for research, reporting, or collaboration

## Contents
- analysis.ipynb: Python notebook with main analysis and plots
- figures/: folder containing generated plots
- report.pdf: summary report of the project

## Usage
1. Open analysis.ipynb in Jupyter Notebook
2. Load your own data files if needed
3. Run the code to reproduce plots and analysis
